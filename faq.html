What are the different ways to select an element in the DOM?

Here are different ways to select an element in the DOM:

  1. 1. getElementById("id")
  2. 2. getElementsByClassName("class")
  3. 3. getElementsByTagName("tag")
  4. 4. querySelector("selector")
  5. 5. querySelectorAll("selector")

What is the difference between innerHTML, innerText, and textContent ?

Here are difference between innerHTML, innerText, and textContent

  1. 1. Use innerHTML when working with HTML markup within elements.
  2. 2. Use innerText to get or set the visible text, considering CSS styling.
  3. 3. Use textContent for a straightforward retrieval or assignment of text content, ignoring HTML tags and CSS styling.

What is event delegation in the DOM?

Event Delegation is a technique in JavaScript where a single event listener is added to a parent element to manage events for its child elements. This approach enhances performance and simplifies event management, especially when dealing with dynamic content.

What is event bubbling in the DOM?

Event bubbling is a DOM event flow where an event starts at the target element and bubbles up to its ancestors. For example, clicking a button inside a div will first trigger the button's click event, then the div's click event, and continue up the DOM hierarchy.
